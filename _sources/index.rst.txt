Konfuzio
========

Our **API Documentation** is available online via https://app.konfuzio.com/api. Have a look at our `YouTube API Tutorial`_ too.

The Konfuzio Software Development Kit, the **Konfuzio SDK**, can be installed via `pip install konfuzio-sdk`_. Find examples in Python and review the source code on GitHub_.

In addition, enterprise clients do have access to our `Python Konfuzio Trainer Module`_ to define, train and run custom Document AI.

Download the **On-Prem and Private Cloud** documentation to deploy Konfuzio on a single VM or on your cluster. Use `this link`_ to access the latest version.

The `Changelog of our Server`_ provides you with insights about any (future) release.
