Our API Documentation is available online via https://app.konfuzio.com/api. Have a look at our YouTube_API_Tutorial too.

The Konfuzio Software Development Kit, the Konfuzio SDK, can be installed via pip_install_konfuzio-sdk. Find examples in Python and review the source code on GitHub.

In addition, enterprise clients do have access to our Python_Konfuzio_Trainer_Module to define, train and run custom Document AI.

Download the On-Prem and Private Cloud documentation to deploy Konfuzio on a single VM or on your cluster. Use this_link to access the latest version.
